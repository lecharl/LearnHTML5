CSS 선택자 1

1. 모든 선택자(*)

		* {
			스타일 속성: 값;
			스타일 속성: 값;
			...
		}

2. 태그 선택자(태그명)

HTML 문서 내에 같은 태그를 모두 선택할 때 사용한다.

		태그명 {
			스타일 속성: 값;
			스타일 속성: 값;
			...
		}

3. 아이디 선택자(#아이디명)

		#아이디명{
			스타일 속성: 값;
			스타일 속성: 값;
			...
		}

4. 클래스 선택자(.클래스명)

		.클래스명{
			스타일 속성: 값;
			스타일 속성: 값;
			...
		}
