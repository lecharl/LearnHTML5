속성 선택자

- 선택자[속성] : 해당 속성을 가진 모든 요소 가져오기
- 선택자[속성=값] : 속성 === 값
- 선택자[속성~=값] : 값에 해당하는 속성이 있으면 선택
- 선택자[속성^=] : 시작값
- 선택자[속성$=] : 끝값
- 선택자[속성*] : 포함
